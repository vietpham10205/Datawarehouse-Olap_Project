LƯU Ý CẤU HÌNH TRƯỚC KHI CHẠY :
Do môi trường máy khác nhau, kính mong Thầy/Cô cập nhật lại Connection Manager trong SSIS:

1. Kết nối Database (OLE DB Source/Destination):
   - Vui lòng đổi Server Name về máy của Thầy/Cô (localhost).
   - Đảm bảo đã Attach file .mdf trong thư mục Database trước đó.

2. Kết nối File nguồn (Excel Connection Manager):
   - File dữ liệu nguồn (csv) em để tại thư mục data
   - Vui lòng Double click vào Connection Manager và trỏ lại đường dẫn đến file trên.
